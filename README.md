# Auth Flow UI

A Next.js project with a complete authentication flow, including login, signup, and forgot password pages.

## Features
- Login Page
- Signup Page
- Forgot Password Page

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
